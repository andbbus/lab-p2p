# Peer-to-Peer Data Science Lab (MVP – Spring 2026)

## Overview

Welcome to the **Peer-to-Peer Data Science Lab – MVP Edition!**

This initiative is designed as a hands-on, collaborative series of workshops geared towards students in Economics & Development (and other social sciences) who want practical skills for conducting impactful research.

**Spring 2026 Features:**
- 6 focused sessions (bi-weekly format)
- Targeted at students working on thesis/research projects
- Peer-to-peer learning approach
- Immediate application to your own work
- Small cohort size (12-15 participants)

---

## Objectives

By the end of the program, participants will have:
- **Research proposal** using structured methodology
- **LaTeX template** for academic writing
- **GitHub portfolio** with version-controlled projects
- **Literature matrix** created with LLM assistance
- **Publication-ready visualizations** for research
- **Interactive dashboard** showcasing findings

---

## Program Structure

**6 Sessions (90 minutes each, bi-weekly):**

### Session 1: Research Project Development
**Focus:** How to develop a research project and address research questions
- Research question formulation (SMART criteria)
- Literature gap identification
- Methodology selection frameworks
- **Output:** One-page research proposal template

### Session 2: LaTeX & Academic Writing (Overleaf)
**Focus:** Professional document preparation
- Document structure and formatting
- Citations and bibliography management
- Mathematical notation and figures
- **Output:** Conference paper template

### Session 3: Version Control with GitHub
**Focus:** Collaborative research workflows
- Repository setup and management
- Branching and merging strategies
- Documentation best practices
- **Output:** Personal research portfolio repo

### Session 4: LLM-Assisted Literature Review
**Focus:** AI tools for research efficiency
- Systematic search strategies
- AI-powered synthesis techniques
- Ethical considerations and citation
- **Output:** Structured literature matrix

### Session 5: Data Visualization with ggplot2
**Focus:** Static publication-ready charts
- Grammar of graphics principles
- Economic data visualization patterns
- Export and formatting for papers
- **Output:** Publication-ready figure set

### Session 6: Interactive Dashboards with Shiny
**Focus:** Dynamic data presentation
- Basic reactive programming
- Economic dashboard design
- Deployment and sharing
- **Output:** Live research dashboard

---

## Session Format

Each session includes:
- **20 minutes:** Concept introduction and theory
- **60 minutes:** Hands-on practical activity
- **10 minutes:** Peer sharing and next steps
- **Take-home assignment** building toward final portfolio
